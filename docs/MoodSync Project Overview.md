# MoodSync Project Overview

## Introduction
MoodSync is a cross-platform mobile application featuring a Kahoot-like style where users join shared sessions to collectively create playlists based on emoji inputs. By aggregating responses, MoodSync generates a playlist that reflects the group's overall mood in a fun, interactive way.

## Vision Statement
MoodSync aims to transform the way groups choose music together, making it a fun, effortless, and inclusive experience where everyone's mood has a voice.

## Motivation & Background
Selecting a song for a group can be difficult and often results in arguments due to differing preferences or "vibes". Traditional methods like rotating an aux cord rarely satisfy the whole group and cause expectations. MoodSync mitigates this by allowing all participants to influence the music in real-time.

## How It Works
1. A Host creates a session and shares a room code.
2. Friends (Joiners) join the room and select moods using an emoji picker.
3. The application averages everyone's mood, and AI generates a prompt to find matching songs via Spotify or Apple Music.
4. The playlist updates dynamically as participants adjust their moods.
5. Users can react to or skip songs within allocated session limits.
