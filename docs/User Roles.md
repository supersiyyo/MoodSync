# User Roles

MoodSync categorizes users into casual group participants and solo listeners.

## Casual Users (Group Setting)
- **Host:** Creates and manages group listening rooms. The Host sets their name, the room name, manages settings, and has the authority to end the session. The Host does not require any active music subscriptions since all media utilizes free 30-second iTunes audio previews.
- **Joiner:** Joins an existing session created by a Host using a room code. Joiners listen in sync, view the current track, and submit emoji reactions to influence the playlist dynamically.
