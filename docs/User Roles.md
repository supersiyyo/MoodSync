# User Roles

MoodSync categorizes users into casual group participants and solo listeners.

## Casual Users (Group Setting)
- **Host:** Creates and manages group listening rooms. The Host sets the room name, manages settings, queues/dequeues songs, and has the authority to end the session. Hosts interact frequently with the emoji system. Full music playback in a session requires the Host to have an active Spotify Premium subscription.
- **Joiner:** Joins an existing session created by a Host using a room code. Joiners listen in sync, chat, view the current queue, and submit emoji reactions to influence the playlist.

## Solo Users
- **Solo User:** Listens to music independently outside of group sessions. They play songs through the app and submit emoji reactions to tailor their own playback without utilizing room management features.
