# Features & Limitations

## Major Features
- **Group Sessions:** Create or join a room with a code, see everyone's emojis, and watch group moods update in real-time.
- **Emoji Picker:** Replaces standard text boxes, offering an array of emojis for users to express their mood.
- **Auto Playlist:** Dynamically generates and updates a Spotify/Apple Music playlist based on combined group moods.
- **Playback Controls:** Hosts can skip songs, manage the queue, and users have limited skips per session to prevent abuse.
- **End-of-Session Stats:** Displays insights such as the most energetic participant, dominant mood, and how collective emojis shaped the playlist.
- **Extra Features:** Save liked playlists, individual song countdown meters, host polling, and a reaction system.

## Scope of Initial Release & Limitations
The initial release will contain all major session features with the following strict limitations:
- **Android Only:** Initially available exclusively on the Android platform via the Google Play Store.
- **Spotify Premium Required:** Full music playback within a session is restricted to Hosts with an active Spotify Premium subscription.
- **Catalog Restrictions:** Song recommendations and playback are limited to the Spotify and Apple Music catalogs. Tracks unavailable on either platform will not be surfaced, and other services like YouTube Music or Tidal are excluded from the initial release.
